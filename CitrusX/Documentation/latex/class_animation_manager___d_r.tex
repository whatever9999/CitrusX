\doxysubsection{Detailed Description}
Enables the triggering of animations on objects around the game using an enum. 

Using the instance of this class you can call \mbox{\hyperlink{class_animation_manager___d_r_aec146fc2c37c8d6fe1c80452468f91b6}{Trigger\+Animation()}} with an enum identifier of a particular animation to make it start. There is also a Fade\+To\+Black() coroutine that will make an animation happen as the screen is fading back in from black to signify time passing.
