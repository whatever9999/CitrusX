\doxysubsection{Detailed Description}
Using an enum and Trigger\+Disturbance(name) an animation/sound will happen to progress the story. 

Move\+Baron(position) is used to set the position of the baron for if he needs to appear somewhere in particular (his original position is still stored) Disappear\+Baron() is a coroutine that prevents him from going to the water bowl and makes him appear idle somewhere for number\+Of\+Seconds\+For\+Baron\+Appearance seconds
