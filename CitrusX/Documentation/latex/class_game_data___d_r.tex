\doxysubsection{Detailed Description}
Holds all variables that need to be saved for the player to restart from their current point in the game. 

Contains variables that need to be saved in order for the game to be loaded in the same state. To be initialised it takes in all the classes that have data that needs to be saved and uses these classes to initialise its variables. \mbox{\hyperlink{class_game_data___d_r}{Game\+Data\+\_\+\+DR}} game\+Data = new \mbox{\hyperlink{class_game_data___d_r}{Game\+Data\+\_\+\+D\+R(player\+Transform)}} -\/$>$ this object will contain the player\textquotesingle{}s current position to be formatted with a binary formatter and written to a file.
