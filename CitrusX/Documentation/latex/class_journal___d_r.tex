\doxysubsection{Detailed Description}
Shows a log of the players activity in a scroll\+Rect and has a list of tasks for the player to complete. 

Use Add\+Journal\+Log(text) to add the next to the top of the scroll\+Rect that shows the journal log. Use Change\+Tasks(tasks) to pass in up to 5 strings in an array to be used as the new tasks. If under 5 are passed in the rest of the tasks will be made blank. Use Tick\+Off\+Task(task\+Number) or Tick\+Off\+Task(task) to tick off the task that corresponds to that number in the list or has the text shown in the task variable. (A literal tick appears next to the task and this is used later to check if the tasks are complete) Use \mbox{\hyperlink{class_journal___d_r_aa5f3937dfb052ba970850768d5aa9e17}{Are\+Tasks\+Complete()}} to check if all tasks have been ticked off. Returns a bool for yes or no.
