\doxysubsection{Detailed Description}
Consists of functions that are added to buttons on a keypad UI that let the player put in numbers, clear input and enter input. 

Number\+Button(number) is placed on a button that is linked to a certain number and the number is an int that pertains to that number. \mbox{\hyperlink{class_keypad_u_i___d_r_af7e56b46b98fae1c5a76d861e3898813}{Enter\+Button()}} is placed on an enter button and will check if the current input aligns with the key\+Pad\+Item\textquotesingle{}s passcode. \mbox{\hyperlink{class_keypad_u_i___d_r_a9c368024cd5ff2999ddd87e83768cb10}{Clear\+Button()}} ensures that the input shown in the input text is blank, along with the string that stores the current input. Clear\+Input() is a coroutine that makes the input flash as a number of Xs to show that the user input the wrong passcode \mbox{\hyperlink{class_keypad_u_i___d_r_aad7e88573bdd92fcd83997d1d9e77340}{Open\+Keypad()}} and \mbox{\hyperlink{class_keypad_u_i___d_r_ae499d1a8c812bfe7f02d25997c99711e}{Close\+Keypad()}} change the lock\+Mode and visibility of the mouse and enable/disable player\+Movement accordingly.
