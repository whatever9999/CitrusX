\doxysubsection{Detailed Description}
Makes a UI image of the baron appear and move towards the camera. 

Use \mbox{\hyperlink{class_baron_popup___d_r_a5c93a27e8f780658ae9fe270fa284ec7}{Spoopy\+Scare()}} to activate the object and play the animation. This starts a coroutine that makes it go away after the length of the animation.
