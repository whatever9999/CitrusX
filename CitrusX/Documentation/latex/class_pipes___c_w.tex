\doxysubsection{Detailed Description}
Each pipe has this script. It knows what the current state of the pipe is and what it should be and allows it to be rotated and checked for its position. 

Get\+Is\+In\+Position() compares the current\+Position of the pipe to the desired\+Position Update() resets the pipes if the player presses X while the UI is open, if it\textquotesingle{}s not complete yet \mbox{\hyperlink{class_pipes___c_w_a77b3c1f8e2bb88f42a2c1959b2474efd}{Rotate()}} moves the UI by 90 degrees and updates the current\+Position of the pipe
