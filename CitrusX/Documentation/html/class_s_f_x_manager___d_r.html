Detailed Description

Instantiates a prefab in the centre of the scene that plays a sound effect.

This class is a don't destroy singleton and should be added to a start scene that will persist throughout the game and can be accessed using SFXManager_DR.instance Sound effects are instantiated using an enum. This is linked to an audio clip passed in through the inspector. The class handles the destuction of the prefab. PlayEffect(name) plays an effect for its length then destroys it PlayEffect(name, secondsToPlay) plays an effect on a loop and destroys it after secondsToPlay
