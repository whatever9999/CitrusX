Detailed Description

Holds all variables that need to be saved for the player to restart from their current point in the game.

Contains variables that need to be saved in order for the game to be loaded in the same state. To be initialised it takes in all the classes that have data that needs to be saved and uses these classes to initialise its variables. GameData_DR gameData = new GameData_DR(playerTransform) -> this object will contain the player's current position to be formatted with a binary formatter and written to a file.
