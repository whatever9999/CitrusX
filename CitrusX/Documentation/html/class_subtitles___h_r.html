Detailed Description

Makes subtitle appear that corresponds to an audio clip from a struct and disappear once the clip has ended.

PlayAudio(id) plays a subtitle according to an enum that links to an audio clip and the text for the subtitle. SubtitleReset(timeToWait) makes the subtitle text blank after timeToWait seconds.
