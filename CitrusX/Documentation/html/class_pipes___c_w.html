Detailed Description

Each pipe has this script. It knows what the current state of the pipe is and what it should be and allows it to be rotated and checked for its position.

GetIsInPosition() compares the currentPosition of the pipe to the desiredPosition Update() resets the pipes if the player presses X while the UI is open, if it's not complete yet Rotate() moves the UI by 90 degrees and updates the currentPosition of the pipe
