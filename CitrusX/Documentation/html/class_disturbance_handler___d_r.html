Detailed Description

Using an enum and TriggerDisturbance(name) an animation/sound will happen to progress the story.

MoveBaron(position) is used to set the position of the baron for if he needs to appear somewhere in particular (his original position is still stored) DisappearBaron() is a coroutine that prevents him from going to the water bowl and makes him appear idle somewhere for numberOfSecondsForBaronAppearance seconds
