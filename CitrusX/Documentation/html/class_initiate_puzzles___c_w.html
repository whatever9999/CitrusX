Detailed Description

Sets up a puzzle when it is inititalised from GameTesting_CW.

InititateSetUpRitualPuzzle() is used at the start of the game In Update monitorInteractions is used to identify what journal logs and tasks are needed and set the state and variables of GOs Initiate<puzzleName>() activates the right GOs for a puzzle to commence
