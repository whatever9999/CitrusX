Detailed Description

Consists of functions that are added to buttons on a keypad UI that let the player put in numbers, clear input and enter input.

NumberButton(number) is placed on a button that is linked to a certain number and the number is an int that pertains to that number. EnterButton() is placed on an enter button and will check if the current input aligns with the keyPadItem's passcode. ClearButton() ensures that the input shown in the input text is blank, along with the string that stores the current input. ClearInput() is a coroutine that makes the input flash as a number of Xs to show that the user input the wrong passcode OpenKeypad() and CloseKeypad() change the lockMode and visibility of the mouse and enable/disable playerMovement accordingly.
