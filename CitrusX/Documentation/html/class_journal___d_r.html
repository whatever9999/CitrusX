Detailed Description

Shows a log of the players activity in a scrollRect and has a list of tasks for the player to complete.

Use AddJournalLog(text) to add the next to the top of the scrollRect that shows the journal log. Use ChangeTasks(tasks) to pass in up to 5 strings in an array to be used as the new tasks. If under 5 are passed in the rest of the tasks will be made blank. Use TickOffTask(taskNumber) or TickOffTask(task) to tick off the task that corresponds to that number in the list or has the text shown in the task variable. (A literal tick appears next to the task and this is used later to check if the tasks are complete) Use AreTasksComplete() to check if all tasks have been ticked off. Returns a bool for yes or no.
